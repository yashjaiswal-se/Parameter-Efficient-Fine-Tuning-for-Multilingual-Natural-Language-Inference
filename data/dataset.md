# Dataset

This project uses the **IndicXNLI** dataset for multilingual Natural Language Inference (NLI). The original dataset is downloaded from Hugging Face and then validated, cleaned, and preprocessed as part of this project. A ready-to-use preprocessed version is also available on Kaggle.

---

# Dataset Sources

## 1. Original Dataset

The original IndicXNLI dataset is available on Hugging Face.

**Source:**
https://huggingface.co/datasets/Divyanshu/indicxnli

IndicXNLI is a multilingual NLI benchmark covering **11 major Indian languages**. Each sample contains a *premise*, a *hypothesis*, and a label indicating their semantic relationship.

Use the original dataset if you want to:

* Reproduce the preprocessing pipeline
* Modify preprocessing steps
* Create custom dataset splits
* Perform your own data analysis

---

## 2. Preprocessed Dataset

A cleaned and processed version of IndicXNLI created for this project is available on Kaggle.

**Source:**
https://www.kaggle.com/datasets/yashjaiswal04518/indicxnli-preprocessed-v1

This version contains the outputs generated by the preprocessing pipeline implemented in this repository.

The preprocessing includes:

* Dataset validation
* Data cleaning
* Text formatting
* Tokenization
* Sequence length analysis
* Truncation analysis
* Nested subset generation

Use the preprocessed dataset if you want to:

* Skip preprocessing
* Start training immediately
* Reproduce the experiments in this repository
* Save preprocessing time

---

# Dataset Structure

Each sample contains three fields.

| Column       | Description                            |
| ------------ | -------------------------------------- |
| `premise`    | Original statement                     |
| `hypothesis` | Statement to compare with the premise  |
| `label`      | Relationship between the two sentences |

---

# Classification Labels

The task is a three-class classification problem.

| Label | Meaning       |
| :---: | ------------- |
|   0   | Entailment    |
|   1   | Neutral       |
|   2   | Contradiction |

---

# Supported Languages

IndicXNLI contains data for the following languages:

* Assamese
* Bengali
* Gujarati
* Hindi
* Kannada
* Malayalam
* Marathi
* Odia
* Punjabi
* Tamil
* Telugu

---

# Dataset Splits

Each language includes three standard splits.

| Split      | Purpose               |
| ---------- | --------------------- |
| Train      | Model training        |
| Validation | Hyperparameter tuning |
| Test       | Final evaluation      |

---

# Validation Pipeline

Before preprocessing, the dataset is validated to ensure its quality.

The validation notebook performs the following checks:

* Dataset loading
* Required columns
* Missing values
* Duplicate records
* Invalid labels
* Class distribution
* Sample inspection

Only datasets that pass these checks are used for preprocessing.
